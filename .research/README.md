# Research

Open-source architectural references for the rebuild.

## Files

- `freecut-notes.md` — architecture notes from FreeCut (MIT).
- `open-source-video-editor-comparison.md` — comparison table covering
  FreeCut, LosslessCut, Remotion, Shotcut, Kdenlive, OpenShot.
- `freecut/` — shallow clone of FreeCut for reference.

## Why FreeCut

FreeCut is the closest open-source analogue to a modern browser video
editor: React + TypeScript + Vite + Zustand + WebCodecs + WebGPU. It
demonstrates the same architectural shape that an independent rebuild of
the target would need.

## License discipline

- **FreeCut**: MIT. Short snippets may be quoted under MIT terms; full
  source is not pasted into the rebuild. Attribution is included.
- **Remotion**: Commercial; reference conventions only, no code.
- **GPL projects**: Reference only; no code lifted.

## What this directory is NOT

- A copy of any project's source.
- A license to redistribute third-party code.
- A substitute for the rebuild's own design.
